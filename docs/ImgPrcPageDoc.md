# ImgPrcPage.vue 기능 구현 사항

## 1. 사이드바 (3탭)

### 필터함수 탭

- 적용할 필텀함수 선택

### Preset 탭

- 저장된 Preset 목록 표시
- Preset 선택 시 캔버스에 로드하여 노드 구성

### 처리목록 탭

- 서버에서 처리된 이미지 목록 표시
- 처리목록 선택 시 캔버스에 로드

---

## 2. 캔버스 영역

### 캔버스 기능

- 줌 in/out
- 노드 표시

### 노드 기능

- disable
- parameter 선택

## 3. 파라미터 영역

- 선택된 노드에 파라미터를 선택한다.
