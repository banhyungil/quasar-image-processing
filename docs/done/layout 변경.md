# 화면 레이아웃

## 1. 사이드바

### 가: 필터함수 탭

- 사이드바에는 카테고리별 필터함수 목록을 볼 수 있는 창
- - 필터함수 카드 구성: 아이콘, 텍스트, 선택 아이콘 버튼, 추가 버튼

### 나. Preset 탭

- Preset 목록 표시
- 클릭 시 Node Flow 로드
  - Preset 기반 로드 후 Preset 저장 시에는 수정이 이루어지도록 한다

### 다. 처리목록 탭

- Preset을 골라서 Node Flow 구성

## 2. Node Flow 섹션

- Node Flow(트리)가 표시된다.
- 원본이미지 선택 버튼
  루트에 원본이미지가 표시된다.
- 현재 Node Flow Preset 저장 기능
- Node 구성
  - 썸네일 이미지
  - 삭제 버튼
  - 파라미터 선택 버튼
  - 헤더 섹션, drag & drop을 통한 노드 이동 기능
