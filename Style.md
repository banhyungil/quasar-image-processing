# CSS / Flex 정리

## flex 축약형

`flex: <flex-grow> <flex-shrink> <flex-basis>`

| 값            | 의미                                          | 예시                                      |
| ------------- | --------------------------------------------- | ----------------------------------------- |
| `flex-grow`   | 남은 공간을 얼마나 **늘어나** 차지할지 (비율) | `1` = 가능한 만큼 늘어남, `0` = 안 늘어남 |
| `flex-shrink` | 공간 부족 시 얼마나 **줄어들**지 (비율)       | `1` = 줄어듦, `0` = 안 줄어듦             |
| `flex-basis`  | 늘어나기/줄어들기 **전 기준 크기**            | `0%`, `auto`, `200px` 등                  |

### 자주 쓰는 패턴

| 값               | 의미                                      |
| ---------------- | ----------------------------------------- |
| `flex: 1 1 0%`   | 남은 공간 균등 분배 (Quasar `col` 클래스) |
| `flex: 0 0 auto` | 크기 고정, 늘어나지도 줄어들지도 않음     |
| `flex: 1 1 auto` | 내용 크기 기준으로 늘어나고 줄어듦        |
| `flex: 1`        | `flex: 1 1 0%` 축약                       |

### `flex-basis: 0%` vs `auto` 차이

```
flex: 1 1 0%   → 내용 크기 무시, 순수 비율로만 공간 분배
flex: 1 1 auto → 내용 크기를 기준으로 남은 공간 비율 분배
```
